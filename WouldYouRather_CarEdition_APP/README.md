# Would You Rather: Car Edition 🚗

A fun React game where you choose between two cars, from budget vehicles to multimillion-dollar hypercars!

## Features

- **46 Cars** ranging from $18k budget cars to $3.2M hypercars
- **5 Categories**: Budget, Economy, Mid-Range, Luxury, Supercar, and Hypercar
- **Score Tracking**: See how many rounds you've completed
- **Progress Bar**: Track your progress through all available cars
- **Beautiful UI**: Modern gradient design with smooth animations

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will open at `http://localhost:3000`

## Game Logic

- Two random cars are presented each round
- Click on the car you'd rather have
- Both cars are removed from the pool after each choice
- Game ends when you've seen all available cars
- Score tracks the number of rounds completed

## Car Categories

- **Budget** ($10k - $30k): Honda Civic, Toyota Corolla, etc.
- **Economy** ($30k - $50k): Mazda CX-5, Subaru Outback, etc.
- **Mid-Range** ($50k - $100k): BMW 3 Series, Tesla Model 3, etc.
- **Luxury** ($100k - $250k): Mercedes S-Class, Bentley Continental GT, etc.
- **Supercar** ($250k - $1M): Ferrari 488, Lamborghini Huracán, etc.
- **Hypercar** ($1M+): Bugatti Chiron, Koenigsegg Jesko, etc.

## Technologies

- React 18
- CSS3 with modern gradients and animations
- Responsive design for mobile and desktop

Enjoy choosing your dream cars! 🏎️
